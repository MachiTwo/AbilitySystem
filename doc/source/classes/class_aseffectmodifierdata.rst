ASEffectModifierData
====================

**Inherits:** ``RefCounted``

Runtime modifier data with custom magnitude support.

.. rst-class:: classref-introduction-group

Description
-----------

ASEffectModifierData extends the basic modifier with runtime-specific features

including custom magnitude overrides. Used by ASEffectSpec during effect execution

to handle dynamic modifier values that may differ from the base definition.



Key features:

- Custom magnitude override capability

- Runtime attribute modification tracking

- Serialization support for state management

- Integration with effect stacking and scaling



Used when effects need dynamic modifier values based on context,

target properties, or other runtime factors.
