ASEventTag
=======

**Inherits:** :ref:`Resource<class_Resource>`

Data-driven payload for transient events.

.. rst-class:: classref-introduction-group

Description
-----------

**ASEventTag** is a transient object used to broadcast signals between components. Unlike Tags, events carry data (instigator, magnitude, payload) and are intended for short-lived occurrences that triggers reactions (e.g., an Ability with :ref:`ASAbility.TRIGGER_ON_EVENT<class_ASAbility_constant_TRIGGER_ON_EVENT>`).
