ASNameTagHistoricalEntry
========================

**Inherits:** ``RefCounted``

Historical entry for NAME tag changes.

.. rst-class:: classref-introduction-group

Description
-----------

ASNameTagHistoricalEntry records when NAME tags (persistent identity tags)

are added or removed from a component. Stores the tag name, target,

timestamp, tick, and whether it was an addition or removal.



NAME tags typically represent persistent states like:

- State.Stunned, State.Stealthed, State.Flying

- Class.Warrior, Class.Mage, Class.Rogue

- Team.Red, Team.Blue, Team.Neutral



Used by the historical query system to track state changes over time,

enable rollback functionality, and support complex game logic that

depends on recent state changes.



Example queries: "Was the player stunned in the last 3 seconds?"

or "How many times did the player enter stealth recently?"
