ASStateCacheEntry
=================

**Inherits:** ``RefCounted``

Lightweight state cache entry for high-performance rollback.

.. rst-class:: classref-introduction-group

Description
-----------

ASStateCacheEntry represents a minimal state snapshot for a specific tick.

Used internally by ASStateCache for fast state restoration during multiplayer rollback.



Stores only essential state information: attributes, tags, and active effects.

Optimized for memory efficiency and fast serialization/deserialization.
