ASDelivery
==========

**Inherits:** :ref:`Node<class_Node>`

Reactive effect delivery node for projectiles and payloads.

.. rst-class:: classref-introduction-group

Description
-----------

**ASDelivery** is a specialized node designed to carry and deliver :ref:`ASPackage<class_ASPackage>`\ s to :ref:`ASComponent<class_ASComponent>` targets.

Unlike :ref:`ASComponent<class_ASComponent>`, which acts as a full ability system hub for an actor, **ASDelivery** is a lightweight utility that can be attached to any node (like a projectile's Area2D) to inject effects upon impact.
