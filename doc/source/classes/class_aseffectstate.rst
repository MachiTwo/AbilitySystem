ASEffectState
=============

**Inherits:** ``RefCounted``

Represents the current state of an active effect.

.. rst-class:: classref-introduction-group

Description
-----------

ASEffectState stores all necessary information about an active effect

for serialization, state management, and rollback purposes.



Used by ASStateCache and ASComponentState to save and restore effect states.

Contains timing information, stack data, and effect level for complete state reconstruction.
