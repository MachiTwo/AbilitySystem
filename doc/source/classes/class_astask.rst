ASTask

**Inherits:** :ref:`RefCounted<class_RefCounted>`

Base class for asynchronous tasks within the Ability System.

.. rst-class:: classref-introduction-group

Description
-----------

**ASTask** is the base class for all asynchronous actions that can be performed during an ability's execution.
