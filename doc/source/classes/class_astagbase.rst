ASTagBase
=========

**Inherits:** ``RefCounted``

Base structure for all Ability System tag types with common functionality.

.. rst-class:: classref-introduction-group

Description
-----------

ASTagBase provides the foundation for all tag types in the Ability System.

It contains common functionality for tag validation, type checking, and utility methods.



This is the base struct that ASNameTag, ASConditionalTag, and ASEventTag inherit from.

It follows the LimboAI bb_param pattern for type-safe tag handling.



All tags must be registered in the AbilitySystem global registry to be considered valid.
