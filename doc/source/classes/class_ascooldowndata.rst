ASCooldownData
==============

**Inherits:** ``RefCounted``

Stores cooldown information for abilities and tags.

.. rst-class:: classref-introduction-group

Description
-----------

ASCooldownData manages cooldown timing and associated tags for abilities

or game mechanics that require time-based restrictions.



Used by ASComponent to track active cooldowns and prevent premature reuse.

Supports both single ability cooldowns and group cooldowns through associated tags.
