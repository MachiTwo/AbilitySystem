ASNameTag
=========

**Inherits:** ``RefCounted``

Persistent identity tag for long-lasting state (State.Stunned, Class.Warrior).

.. rst-class:: classref-introduction-group

Description
-----------

ASNameTag represents persistent identity or state tags that remain until explicitly removed.

Used for character states (State.Stunned, State.Dead), class affiliation (Class.Warrior),

team membership (Team.Blue), and other long-term identity markers.



These tags form the foundation of an actor's persistent identity and are typically

managed through ASComponent's add_tag/remove_tag methods.



From BUSINESS_RULES.md: "Represent persistent identity or state."
