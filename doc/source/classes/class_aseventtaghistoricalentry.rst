ASEventTagHistoricalEntry
=========================

**Inherits:** ``RefCounted``

Historical entry for event occurrences.

.. rst-class:: classref-introduction-group

Description
-----------

ASEventTagHistoricalEntry stores a complete event occurrence in the

historical buffer. Contains the full event data plus the tick when

it was recorded, enabling precise event tracking and rollback.



Used by ASComponent's event history buffer to maintain a chronological

record of all events for analysis, debugging, and multiplayer

synchronization purposes.



Combined with the historical query APIs, enables powerful event

analysis like "did damage occur in the last 2 seconds" or

"how many times was this ability activated recently".
