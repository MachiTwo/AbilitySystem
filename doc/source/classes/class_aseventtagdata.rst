ASEventTagData
==============

**Inherits:** ``RefCounted``

Complete data for event dispatching.

.. rst-class:: classref-introduction-group

Description
-----------

ASEventTagData contains all information needed to dispatch and track

events in the Ability System. Includes event identification, participants,

magnitude values, custom payload data, and timing information.



Key features:

- Event identification through event_tag

- Participant tracking (instigator and target)

- Magnitude values for quantitative events

- Custom payload for flexible data attachment

- Timestamp and tick tracking for historical purposes

- Node reference helpers for easy access



Used by ASComponent for event dispatching and by the historical

system to track event occurrences for rollback and analysis.
