ASConditionalTag
================

**Inherits:** ``RefCounted``

Requirement/block tag for ability prerequisites and immunities (Can.Parried, Immune.Fire).

.. rst-class:: classref-introduction-group

Description
-----------

ASConditionalTag represents tags used in logical evaluation for ability requirements

and effect blockers. These are typically short-term tags that grant permissions or

special conditions.



Used for ability prerequisites (Can.Parried, Can.Dodged), damage immunity

(Immune.Fire, Immune.Poison), and state conditions (State.Grounded, State.Flying).



From BUSINESS_RULES.md: "Used in logical evaluation (AND/OR) for requirements."

These tags participate in Required All/Any and Blocked All/Any predicates.
