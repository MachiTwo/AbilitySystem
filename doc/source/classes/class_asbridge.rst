ASBridge
========

**Inherits:** ``RefCounted``

Integration bridge between Ability System and LimboAI.

.. rst-class:: classref-introduction-group

Description
-----------

The **ASBridge** provides seamless integration between Ability System (AS) and LimboAI Behavior Trees / Hierarchical State Machines. It auto-detects LimboAI at runtime and registers custom BT tasks and LimboStates that interact with :ref:`ASComponent<class_ASComponent>`.

This bridge is optional - if LimboAI is not present, the bridge initializes but remains inactive without errors.
