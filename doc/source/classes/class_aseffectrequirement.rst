ASEffectRequirement
===================

**Inherits:** ``RefCounted``

Attribute requirement for effect activation.

.. rst-class:: classref-introduction-group

Description
-----------

ASEffectRequirement defines conditions that must be met for an effect

to activate or apply. Typically used for attribute-based requirements

like minimum health levels, resource thresholds, or stat checks.



Used by ASEffect to specify activation conditions and by the effect

system to validate whether requirements are satisfied before applying

effects to targets.



Requirements are evaluated during effect activation and can be combined

with tag-based requirements for complex activation logic.
