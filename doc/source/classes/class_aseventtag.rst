ASEventTag
==========

**Inherits:** ``RefCounted``

Event dispatcher tag for transient system notifications (Event.Damage, Event.Ability.Activated).

.. rst-class:: classref-introduction-group

Description
-----------

ASEventTag represents event identifiers used with the dispatch system. These are

transient signals that flow through the system informing about specific occurrences.



Events are instant messengers that carry context including instigator, magnitude,

and custom payload. They prevent spaghetti code by providing a clean event-driven

architecture for combat, ability triggers, and system notifications.



From BUSINESS_RULES.md: "Represent specific informative occurrences."

Events are dispatched via ASComponent.dispatch_event() and can be queried

through the Events Historical buffer for temporal reactivity.



ASEventTag provides convenience methods for dispatching events and checking

if they occurred recently.
