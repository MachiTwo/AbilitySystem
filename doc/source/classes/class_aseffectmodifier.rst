ASEffectModifier
================

**Inherits:** ``RefCounted``

Represents a single attribute modifier for effects.

.. rst-class:: classref-introduction-group

Description
-----------

ASEffectModifier defines how an effect modifies a specific attribute.

Contains the attribute name, operation type (add, multiply, set, etc.),

and magnitude value for the modification.



Used by ASEffect to define attribute changes and by ASEffectSpec

for runtime modifier calculations. Supports serialization for

save/load operations and network replication.
