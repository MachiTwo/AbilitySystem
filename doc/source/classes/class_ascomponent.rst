ASComponent
===========

**Inherits:** :ref:`Node<class_Node>`

Core component for managing abilities, attributes, and effects.

.. rst-class:: classref-introduction-group

Description
-----------

The **ASComponent** (ASC) is the central hub for an actor's logic. It handles the lifecycle of abilities, applies effects, and tracks attribute values. Must be a child of :ref:`CharacterBody2D<class_CharacterBody2D>` or :ref:`CharacterBody3D<class_CharacterBody3D>`.
