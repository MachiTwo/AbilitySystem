ASAttributeValue
================

**Inherits:** ``RefCounted``

Stores base and current values for attributes.

.. rst-class:: classref-introduction-group

Description
-----------

ASAttributeValue maintains both the base value and current modified value

of an attribute. The base value represents the original value before

modifiers, while the current value reflects all active modifications.



Key features:

- Base value preservation for modifier calculations

- Current value tracking for runtime attribute access

- Difference calculation for effect magnitude

- Serialization support for state management



Used by ASAttributeSet to store attribute values and by the effect

system to calculate and apply attribute modifications.
