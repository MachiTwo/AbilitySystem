ASComponentState
================

**Inherits:** ``RefCounted``

Complete state representation of an ASComponent for serialization.

.. rst-class:: classref-introduction-group

Description
-----------

ASComponentState provides a comprehensive snapshot of an ASComponent's

entire state, including historical data. Designed for save/load operations,

full serialization, and complete state transfer between game sessions.



Unlike ASStateCache which stores minimal data for performance-critical rollback,

ASComponentState captures the complete component state including:

- All attributes and their values

- All tags currently owned

- All active effects with full state

- All cooldowns and remaining times

- Complete historical buffers for all tag types



Used by save systems, network replication, and debugging tools.
